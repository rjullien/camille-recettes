# 🍽️ Les Recettes de Camille

Un site web élégant pour partager les délicieuses recettes de Camille, avec un design inspiré de ses créations Canva.

## 🌟 Caractéristiques

- **Design authentique** : Style Canva avec palette beige/nature
- **Responsive** : Optimisé pour mobile et cuisine
- **Recherche intelligente** : Filtrage par nom, ingrédients et catégories
- **Impression parfaite** : Mise en page A4 optimisée pour l'impression
- **Navigation intuitive** : Interface claire et moderne

## 🎨 Palette de couleurs

- **Brun foncé** (`#3D2B1F`) - Texte principal
- **Beige clair** (`#F5F0E8`) - Fond principal  
- **Beige sable** (`#E8DCC8`) - Éléments et accents
- **Vert doux** (`#7A8471`) - Accents végétaux

## 📱 Fonctionnalités

### Page d'accueil
- Barre de recherche temps réel
- Filtres par catégorie
- Grille responsive des recettes
- Animation smooth des cartes

### Pages de recettes
- Layout A4 portrait (style Canva)
- Section ingrédients avec fond beige
- Instructions numérotées étape par étape
- Bouton d'impression optimisé
- Conseils et astuces de Camille

### Technologies
- HTML5 sémantique
- CSS3 avec variables et Grid/Flexbox
- JavaScript vanilla (pas de frameworks)
- Google Fonts (Pacifico + Lato)
- Progressive Web App ready

## 🖨️ Impression

Le site inclut des styles d'impression parfaitement optimisés :
- Format A4 portrait automatique
- Une recette = une page exactement
- Conservation de la palette de couleurs
- Mise en page fidèle au design Canva
